# Fetch Replacer

A userscript that allows you to intercept fetch() calls and serve your own version of the requested resource.

# Installation

1. Install a userscript manager extension like [Tampermonkey](https://www.tampermonkey.net/) in your browser

2. [![Button Icon](https://img.shields.io/badge/Click%20to%20install%20Fetch%20Replacer-3944BC?style=flat&logoColor=white&logo=Files)](https://github.com/mmistika/fetch-replacer/raw/refs/heads/main/fetch-replacer.user.js)

# Usage

1. Open the userscript editor
2. Specify a substring to detect the resource to replace
3. Specify the URL of your replacement resource 
4. Trigger a fetch() call
